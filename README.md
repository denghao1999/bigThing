# bigThing

## 运行贪吃蛇

这个仓库里新增了一个可在浏览器运行的贪吃蛇页面：`home/snake.html`。

### 本地启动（推荐）

在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后浏览器打开：

- `http://localhost:8000/index.html`（左侧菜单：**小游戏 → 贪吃蛇**）
- 或直接打开 `http://localhost:8000/home/snake.html`

### 操作

- **方向键 / WASD**：移动
- **空格**：暂停/继续
- **R**：重新开始
- 手机可用页面底部方向键，或在画布上滑动改变方向（轻点画布可暂停/继续）
